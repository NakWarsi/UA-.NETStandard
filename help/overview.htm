Unified Architecture Technology Sample Applications

Sample Applications Overview

The OPC Foundation strongly encourages the use of Certified commercial products and development tools from member companies.

The OPC Foundation provides several application samples to demonstrate the use of OPC UA technology, including:

  • Welcome Dashboard

  • Configuration Utility: For configuring UA application security, and COM/Interoperability etc.

  • Discovery Server: For maintaining a list of UA applications that are currently available and online.

  • Generic UA Client: A general purpose Client.

  • Generic UA Server: A general purpose Server.

  • Data Access Technology Samples: See Data Access Overview.

    • Data Access Server: A Server designed to showcase the Data Access feature/functionality.

    • Data Access Client: A Client designed to work with the Data Access Server.

  • Historical Access Technology Samples: See Historical Access Overview.

    • Data Server: A Server designed to showcase Historical Data feature/functionality.

    • Data Client: A Client designed to work with the Historical Data Server.

    • Events Server: A server designed to showcase the Historical Events feature/functionality.

    • Events Client: A Client designed to work with the Historical Events Server.

  • Alarms & Conditions Technology Samples: See Alarms & Conditions Overview.

    • Alarms Conditions Server: A Server designed to showcase the Alarms & Conditions feature/functionality

    • Alarms Conditions Client: A Client designed to work with the Alarms Conditions Server.

These reference implementations are built using a common infrastructure (UA .NET API) enabling them to share the same characteristics. The UA Configuration Tool provides screens to help define and manage these applications, and any other applications that are built upon the same infrastructure.
