UA Configuration Tool Overview

The .NET Stack and subsequent API provides an infrastructure suitable for OPC UA applications development, deployment, and management. The Configuration Tool serves as a central management application providing many administrative features:

  • Application Configuration

  • Certificate Management for an Application

  • Certificate Repository Management

  • COM Interoperability Management
